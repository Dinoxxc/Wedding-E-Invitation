# Background Music

Place your wedding background music file here.

## Recommended Format
- MP3 format (best browser compatibility)
- File name: `wedding-music.mp3`
- Recommended: Instrumental or soft romantic music
- Duration: 3-5 minutes (the audio will loop automatically)

## Free Music Resources

1. **YouTube Audio Library** - Free, no attribution required

## How to Add Your Music

1. Download or obtain your music file
2. Convert to MP3 if necessary
3. Rename to `wedding-music.mp3`
4. Place in this folder (`frontend/audio/`)
5. The website will automatically play it on page load

## Browser Autoplay Policy

Note: Most modern browsers block autoplay with sound until the user interacts with the page. The music player is configured to:
1. Attempt autoplay on page load
2. Start playing after the first user click if autoplay is blocked
3. Provide a toggle button for manual control

## File Not Found?

If you don't add a music file, the player will still appear but won't play anything. The website will work fine without music.
